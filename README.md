# Benefits App

A GitHub-ready front-end prototype containing two healthcare literacy tools:

## 1. BenefitSmart
An interactive learning app for high school students, employees, and patients.

Topics:
- Premium
- Deductible
- Copay
- Coinsurance
- HSA/FSA
- In-network vs out-of-network
- Prior authorization
- Out-of-pocket maximum

Includes:
- Learning style selector
- Visual, audio, reading, hands-on, and gamified modes
- Scenario simulator
- Progress dashboard concept

## 2. Benefits Translator
A healthcare cost and benefits translator.

Demo features:
- Procedure cost estimator
- Deductible remaining calculator
- Coinsurance estimator
- Prior authorization flag
- Cheaper alternative suggestions
- Network tips

## 3. AI Navigator
A mock AI chatbot that explains insurance terms in plain English.

## Important
This is a front-end prototype. It does not yet connect to:
- real insurer APIs
- hospital systems
- Epic/Cerner
- real claims databases
- OpenAI API
- Firebase Authentication

## How to deploy on GitHub Pages
1. Upload `index.html`, `style.css`, and `script.js` into the root of your repository.
2. Go to Settings → Pages.
3. Choose Deploy from Branch.
4. Select `main` and `/root`.
5. Save.
6. Open your GitHub Pages link.
