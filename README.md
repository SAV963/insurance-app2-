# BenefitSmart + Benefits Translator Prototype

Open `index.html` in a browser.

Includes:
- BenefitSmart education app prototype
- Learning-style selection
- Insurance literacy modules
- Real-life scenario simulator
- Benefits Translator cost estimator prototype
- Deductible, coinsurance, prior authorization, and alternatives output

This is a front-end prototype using sample pricing logic. A real production app would need insurer APIs, provider directories, CPT/code mapping, CMS price transparency data, HIPAA/security review, and legal review.
